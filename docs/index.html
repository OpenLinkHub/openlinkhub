连接设备、传感器、数据与规则

OpenLinkHub IoT Core 提供设备管理、传感器绑定、HTTP 数据上报、时序数据存储、 规则告警、状态查看和历史数据导出能力，帮助开发者快速构建设备互联与数据协同平台。

设备与传感器

管理设备基础信息，绑定温度、湿度、电量、GPS 等多类传感器。

时序数据

基于 TimescaleDB 存储遥测数据，支持按设备、传感器和时间范围查询。

规则告警

配置阈值规则，数据上报时自动判断并生成告警。

Web 控制台

提供列表、筛选、分页、弹窗编辑、状态查看和 CSV 导出。

快速开始

PGPASSWORD=iot_password psql -h localhost -p 5432 -U iot_user -d postgres -f deploy/create-database.sql

cd backend
mvn spring-boot:run

cd ../console
npm install
npm run dev
